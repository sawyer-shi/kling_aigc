# 可灵 AIGC

一个基于可灵 AI 3.0 模型的 Dify 插件，提供视频、图像与主体（元素）生成能力。支持 Omni 视频、文生视频、图生视频、Omni 图像、图像生成以及主体管理与任务查询。

## 版本信息

- **当前版本**: v0.0.1
- **发布日期**: 2026-02-16
- **兼容性**: Dify 插件框架
- **Python 版本**: 3.12

### 版本历史
- **v0.0.1** (2026-02-16): 初始发布，包含可灵 3.0 视频/图像/主体工具

## 快速开始

1. 在 Dify 环境中安装插件
2. 配置可灵 AI 凭证（Access Key + Secret Key）
3. 创建任务并查询结果

## 核心特性

- **视频生成**: Omni 视频、文生视频、图生视频
- **图像生成**: Omni 图像与图像生成任务
- **主体管理**: 创建、查询、删除主体
- **任务查询**: 单任务查询，支持自动下载
- **水印控制**: 可选输出水印版本

## 核心功能

### 视频工具

- **Omni 视频创建**: 单镜头/多镜头视频生成
- **文生视频**: 文本提示词生成视频
- **图生视频**: 首尾帧与蒙版控制
- **视频查询**: 状态、结果链接与自动下载

### 图像工具

- **Omni 图像创建**: 单图/组图生成
- **图像生成**: 文本与可选参考图
- **图像查询**: 状态、结果链接与自动下载

### 主体工具

- **主体创建**: 图像/视频参考主体
- **主体查询**: 结果状态与主体信息
- **主体删除**: 删除自定义主体

## 依赖要求

- Python 3.12
- Dify 平台访问权限
- 可灵 AI 凭证（Access Key + Secret Key）
- 依赖包（通过 requirements.txt 安装）:
  - dify_plugin>=0.2.0
  - requests>=2.31.0
  - PyJWT>=2.8.0
  - Pillow>=9.0.0

## 安装与配置

1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
2. 在插件设置中配置可灵 AI 凭证：
   - **Access Key**
   - **Secret Key**
3. 在 Dify 环境中安装插件

## 说明

- 生成内容默认保留 30 天，请及时下载
- 查询工具支持自动下载选项
