## English Version

### Overview

The Kling AIGC Plugin ("Plugin") provides AI-powered video, image, and element generation using Kling AI 3.0 models. This privacy policy explains how we handle your data when you use this plugin within the Dify platform.

### License Information

This plugin is licensed under Apache License 2.0, a permissive open-source license that allows use, modification, and distribution. The full license text is available in the LICENSE file included with this plugin.

### Data Collection

**What we DO NOT collect:**
- Personal identification information
- User account details
- Usage analytics or tracking data
- Device information
- Location data
- Cookies or similar tracking technologies

**What we process:**
- Text prompts for video and image generation
- Image files for image-to-video, omni-image, and element creation
- API credentials (Access Key and Secret Key) for Kling AI services
- Task IDs for querying generation results

### Data Processing

**External API Processing:**
- All generation requests are sent to Kling AI API services
- Text prompts and image files are transmitted to Kling AI servers for AI processing
- API credentials are used to authenticate requests to Kling AI services
- Generation tasks are processed asynchronously on Kling AI infrastructure

**Local Processing:**
- Plugin configuration and settings are stored locally in Dify
- Temporary files may be created during file processing
- No permanent data storage beyond the plugin session

### Data Storage

**No Local Data Retention:**
- We do not store copies of your text prompts after generation
- We do not store copies of your uploaded images after processing
- We do not retain generated images or videos locally
- API credentials are stored securely in Dify's plugin configuration

**Kling AI Data Handling:**
- Kling AI processes your prompts and images according to their privacy policy
- Generated content is stored temporarily on Kling AI servers for retrieval
- Users should review Kling AI's privacy policy for data handling practices

### Security Measures

**Data Protection:**
- API credentials are transmitted securely over HTTPS
- All communication with Kling AI API uses encrypted connections
- Plugin operates within Dify's secure environment
- No unauthorized network connections are made

**Third-Party Services:**
- This plugin relies on Kling AI API services
- Users must have valid Kling AI API credentials
- All AI processing occurs on Kling AI infrastructure

## 中文版本

### 概述

可灵 AIGC 插件（"插件"）使用可灵 AI 3.0 模型提供视频、图像和主体生成能力。本隐私政策说明您在 Dify 平台内使用插件时，我们如何处理您的数据。

### 许可证信息

本插件采用 Apache License 2.0 许可证，这是一个宽松的开源许可证，允许使用、修改和分发软件。完整许可证文本见插件内的 LICENSE 文件。

### 数据收集

**我们不收集的内容：**
- 个人身份信息
- 用户账户详细信息
- 使用分析或跟踪数据
- 设备信息
- 位置数据
- Cookie 或类似跟踪技术

**我们处理的内容：**
- 视频与图像生成的文本提示词
- 用于图生视频、Omni 图像与主体创建的图像文件
- 可灵 AI 凭证（Access Key 与 Secret Key）
- 任务查询所需的任务 ID

### 数据处理

**外部 API 处理：**
- 所有生成请求发送到可灵 AI API 服务
- 文本提示词与图像文件传输到可灵 AI 服务器进行处理
- API 凭证用于认证请求
- 生成任务在可灵 AI 基础设施上异步处理

**本地处理：**
- 插件配置与设置存储在 Dify 本地
- 文件处理期间可能会生成临时文件
- 除插件会话外不进行永久性数据存储

### 数据存储

**不保留本地数据：**
- 生成后不保留文本提示词副本
- 处理后不保留上传图片副本
- 不在本地保留生成的图像或视频
- API 凭证安全存储在 Dify 的插件配置中

**可灵 AI 数据处理：**
- 可灵 AI 按其隐私政策处理提示词与图像
- 生成内容在可灵 AI 服务器上临时存储以供检索
- 建议用户查看可灵 AI 隐私政策了解数据处理方式

### 安全措施

**数据保护：**
- API 凭证通过 HTTPS 安全传输
- 与可灵 AI API 的通信使用加密连接
- 插件运行于 Dify 的安全环境
- 不进行未经授权的网络连接

**第三方服务：**
- 插件依赖可灵 AI API 服务
- 用户需具备有效的可灵 AI 凭证
- AI 处理在可灵 AI 基础设施中完成
